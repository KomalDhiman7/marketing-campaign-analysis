## Overview
This project analyzes marketing campaign performance using Python. The goal is to identify high-performing campaigns and customer segments.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Campaign type vs conversion rate
- Channel performance analysis
- ROI and cost analysis
- Customer segmentation
- Engagement impact on conversion

## Key Insights
- Email campaigns showed higher conversion rates
- Google Ads performed better among channels
- Higher engagement score led to better conversion
- Some campaigns had high cost but low ROI
